# 🐍⚡ Cyber Snake Game

A modern cyberpunk-themed Snake game with neon visuals and synthwave aesthetics.

![Cyber Snake Preview](https://img.shields.io/badge/Status-Development%20in%20Progress-blue)
![GitHub](https://img.shields.io/github/license/fandiutomoputra/cyber-snake-game)
![GitHub last commit](https://img.shields.io/github/last-commit/fandiutomoputra/cyber-snake-game)

## 🎮 Features
- **Cyberpunk neon blue theme** with glitch effects
- **Smooth 60 FPS gameplay** with HTML5 Canvas
- **Keyboard + touch controls** for mobile/desktop
- **Local high score tracking** with LocalStorage
- **Progressive difficulty** - speed increases with level
- **Responsive design** - play on any device

## 🚀 Quick Start

### **Option 1: Live Demo (Recommended)**
Play directly in your browser:
```
https://fandiutomoputra.github.io/cyber-snake-game
```

### **Option 2: Local Play (Simple)**
1. Download the project ZIP
2. Extract to any folder
3. Double-click `index.html`
4. Game opens in your browser!

### **Option 3: Local Server**
```bash
# 1. Clone the repository
git clone https://github.com/fandiutomoputra/cyber-snake-game.git

# 2. Navigate to folder
cd cyber-snake-game

# 3. Run local server (Python)
python -m http.server 8000

# 4. Open browser to:
# http://localhost:8000
```

## 🎯 How to Play

### **Controls:**
- **Arrow Keys** ←↑↓→ : Move the snake
- **Spacebar** : Pause/Resume game
- **R Key** : Restart game
- **M Key** : Toggle music (placeholder)
- **Touch Swipe** : Mobile controls (swipe direction)

### **Game Rules:**
1. Guide the snake to eat the pink food (apple)
2. Each food increases your score by 10 × level
3. Avoid hitting walls or yourself
4. Game speeds up every 50 points (level up)
5. Try to beat your high score!
